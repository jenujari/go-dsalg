# Array

---

Array is collection of item, It has index and value at that index.

Array has two types

1. Static Array
2. Dynamic Array

Static Array has fixed size, Dynamic array can increase size at runtime.

Array has following Oprations

- Push O(1)
- Pop O(1)
- Lookup O(1) -- get by index
- Tranvers/Search O(n)
- Insert O(n)
- Delete O(n)
